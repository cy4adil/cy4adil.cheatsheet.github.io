CSS Selectors

Selectors Name Description
h1 Type Selection This way of selection is known as selection by type. Here, we will use html element's tag name to select the specific element to apply the CSS properties.
.heading Class Selection This selection is called selection by class. It starts with dot and then followed by the name of the class assigned to the specific html element. One class can be assigned to multiple html elements.
#specialHeading Id Selection This seleciton is called selection by id. It starts with number/hash sign and then followed by the name of the id. Every html tag/element can have only one unique id, we cannot assign one id to multiple html elements. So, id is always unique and points towards only one html element.

CSS Selector's Specificity

Selectors Name Description
#specialHeading Id Selection The specificity of id selector will always be on priority by the browser. So, the id selector has more proximity.
.heading Class Selection The specificity of class selector comes after the id selector. if there is no id defined for the specific element then class attribute will be given a priority and CSS ruleset defined in the class selector will be applied to that specific html element.
h1 Type Selection The specificity of type selector comes after the id and class selector. If there is no id or class selection is defined then browser will consider type selection if it is defined or mentioned.
